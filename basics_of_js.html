Javascript Basics

Data Types :

  • Numbers - Numeric Value
  • String - Text Sequence
  • Boolean - True or False
  • Null - Empty Value
  • Undefined - Unknown Type

Variables :

  • Var ( Funtion scope) : Flexible Variable
  • Let ( Block scope ) : Block Variable
  • Const ( Block scope) : Constant Variable

Synchronous :

  • Code executes in order, one line at a time
  • Waits for previous task to complete
  • Blocking, can cause delays

Asynchronous :

  • Code executes simultaneously, multiple tasks
  • Doesn't wait for previous task to complete
  • Non-blocking, efficient and scalable
